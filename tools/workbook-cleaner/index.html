Excel Data Cleanup Tool

Data from multiple sources has grown organically. Duplicate IDs, inconsistent formats, missing required fields. Every analysis starts with an hour of cleanup before the real work begins.

The Workbook Cleaner ingests raw data files from any source, applies configurable column-level validation rules, eliminates duplicates, and delivers a single analysis-ready master file with a complete fix changelog.

Ask about early access See the service

Planned capabilities

  • Drop-in support for CSV, XLSX, and TSV sources
  • Column-level rule editor for non-technical users
  • Duplicate detection across configurable key sets
  • Merge strategy per field: keep first, keep last, concatenate, numeric max
  • Full changelog showing every row that was fixed, merged, or dropped
  • Output in the same format as the input, or a normalized master layout

Status

The underlying engine is stable and has been used inside other tools on this site. The standalone version is in active development. The rules editor for non-technical users is being refined based on feedback from the teams piloting it.

Why it exists

The VNE tool solves one specific shape of cleanup: name matching. The Workbook Cleaner is the generalist. Different rules, different column types, same end goal. A single analysis-ready master file.
