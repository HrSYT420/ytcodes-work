WPR Live
☑
Batch Validation Pipeline
// five checks, one click

Runs a structured multi-step validation pipeline on any incoming data batch. Each step is independent. Outputs a colour-coded review package flagging mismatches, inactive records, and unrecognized entries.

multi-step pipeline reconciliation audit trail
View details ⌄
The Pipeline
  • Import: Load input batches in multiple formats into dedicated tabs
  • Registry Rebuild: Reconstruct a current reference registry from raw extracts
  • Fuzzy Name Match: Match batch names to registry, surfacing near-duplicates
  • ID Cross-check: Verify each record's identifier against the registry
  • Inactive Check: Flag any records belonging to deactivated entries
  • Steps are independent — re-run any single step without restarting the pipeline
  • Archivable, colour-coded results package with full run log
Use case

Any quality-review workflow over structured batches: payment review, data reconciliation, record validation against a reference set, import auditing, compliance checks.
